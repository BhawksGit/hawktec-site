Quick Start

  1. Download the latest firmware from lnh-team.org.
  2. Format the included 8 GB micro SD as FAT32 if needed.
  3. Extract firmware files onto the root of the micro SD.
  4. Insert micro SD into the DSPico, insert the cart into your DS.
  5. Power on — Pico Launcher should appear.

Compatibility

Works in both DS and DSi modes:

DSDS LiteDSi 2DS3DSNew 2DS XLNew 3DS XL

With DSi-mode support via WRFUxxed, the DSPico runs on unmodified DSi and 3DS consoles with full DSi-exclusive access.

FAQ

Is this fully open-source?

Yes. Hardware, firmware, software, casing, and artwork are all open-source via the LNH Team.

Can I update firmware over USB?

Yes — the DSPico supports USB firmware updates.

Pico Loader vs Pico Launcher?

Pico Loader handles booting and console detection. Pico Launcher is the menu you use to browse and launch content. Both are included.
