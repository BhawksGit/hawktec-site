Quick Start

📌

To flash up-to-date firmware or use something other than the pre-flashed WRFUxxed kernel that comes with the HawkTec DSPico, follow this guide on YouTube. This Reddit discussion is also helpful for setting up the firmware .uf2 file.

  1. Download the HawkTec DSPico filesystem and extract it onto a FAT32-formatted micro SD:
    Download HawkTec Pico Launcher FilesystemVersion HWKTC_DSPICO_V1.0.0
  2. Insert micro SD into the DSPico, insert the cart into your DS.
  3. Power on — Pico Launcher should appear.

SD Card Requirements

The micro SD card must be formatted as FAT32. Cards 32 GB or smaller can be formatted directly through your operating system. Cards larger than 32 GB require a third-party formatting tool on Windows — Rufus is a commonly recommended option.

Expected File Structure

After extraction, your SD root should look like:

microSD/ ├── _pico/ ├── GBA ROMS/ ├── GBC ROMS/ ├── NDS ROMS/ ├── NES ROMS/ ├── shared1/ ├── shared2/ ├── sys/ ├── _picoboot.nds └── README
⚠

These files must be in the root directory of your micro SD card, not inside any folders. Often, these files end up in a folder (like "HWKTC_DSPICO_V1.0.0") when extracting the kernel files. Move them from the folder to the main directory of the micro SD card to resolve the issue.

FAQ

Is this fully open-source?

Yes. Hardware, firmware, software, casing, and artwork are all open-source via the LNH Team.

Can I update firmware over USB?

Yes — the DSPico supports USB firmware updates.

Pico Loader vs Pico Launcher?

Pico Loader handles booting and console detection. Pico Launcher is the menu you use to browse and launch content. Both are included.
